# PlayerReports PAPI Expansion

PlaceholderAPI expansion for PlayerReports.

## Placeholder

```text
%playerreports_reports%
```

Aliases:

```text
%playerreports_count%
%playerreports_total%
%playerreports_open%
```

Debug:

```text
%playerreports_status%
```

## Build on GitHub

Upload this folder to a GitHub repository. Open the Actions tab and run the build workflow.
The compiled jar will be attached as a workflow artifact.

## Install

Put the compiled jar into:

```text
plugins/PlaceholderAPI/expansions/
```

Then run:

```text
/papi reload
```
